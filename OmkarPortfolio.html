TIN is an NSDL e-Governance project which is the first electronic securities depository in India with national coverage of 1.4 Cr clients. TIN is an initiative by Income Tax Department of India (ITD for collection, processing, monitoring and accounting of direct taxes. Designed and developed Hybrid application in PAAM module and provided production support for the existing Web Application.

Skills Used: Spring MVC 3.0, Hibernate 3.0, EJB 3.0, IBM DB2, Shell Script, JQuery, JSP, JavaScript, Ajax, Web Services.

Skills Used:ASP.Net, C#, SQL server 2012.
